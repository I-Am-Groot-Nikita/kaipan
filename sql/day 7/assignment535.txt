Q-1 Write a query which will print name of each student, and his or her attendance report for current month. 
Fields : Name, Attendance Report for Current Month

	Under "Attendance Report for Current Month" column:
		if 'studentpresent.present' is less than or equal to 7 print "Poor", 
		if 'studentpresent.present' is more than 7 and less than or equals 15 print "Good", 
		if 'studentpresent.present' is more than 15 and less than or equals 20 print "Better", 
		if 'studentpresent.present' is more than 20 print "Best".

select s.name,
case
    when sp.present <= 7 then 'Poor'
    when sp.present > 7 and sp.present <= 15 then 'Good'
    when sp.present > 15 and sp.present <= 20 then 'Better'
    else 'Best'
end as Attendance_Report
from student_trainee_35 s right join studentpresent_trainee_35 sp on s.recid = sp.recid
where sp.month_ = month(now()) ;



Q-2 Write a query to print student name, total number of hours for Jan, Feb, March month. Information for each student should be printed in a single row.

	Fields : Name, Working Hours for Jan, Working Hours for Feb, Working Hours for March

	See following example:
	Name		Working Hours for Jan	Working Hours for Feb	Working Hours for March
	Minesh		80			70			90
	Reena		70			80			90


select distinct s.recid,s.name "Name",
case
    when month(sh.datex)=1 then sh.hours end as Jan,
case
    when month(sh.datex)=2 then sh.hours end as Feb,
case
when month(sh.datex)=3 then sh.hours end as March
from student_trainee_35 s
inner join studenthours_trainee_35 sh on s.recid = sh.recid;

Q-5 Print information for the student having presence for Jananuary between 20 to 25.
	Fields: Name, Standard, Present

select s.name "Name", sd.standard "Standard" ,sp.present "Present"
from student_trainee_35 s
inner join studentdetail_trainee_35 sd on s.recid = sd.recid
inner join studentpresent_trainee_35 sp on s.recid = sp.recid
where sp.month_ = 1 and sp.present between 20 and 25;

Q-9 Write a query which will print name of each student, and his or her attendance report for current month. 
	Fields : Name, Attendance Report for Current Month

	In "Attendance Report for Current Month" column 
	if student has presence for that month then print "Present"
	else print "Not Present".

select s.name,
case
    when sp.present > 0 then 'present'
    else 'not present'
end as Attendance_Report
from student_trainee_35 s inner join studentpresent_trainee_35 sp on s.recid = sp.recid
where sp.month_ = month(now()) ;

Q-11 Print the student information having average presence (in percentage) for Jan, Feb and March between 60 to 75.
	Fields: Name, Standard, Present

select s.name, sd.standard, round((avg(sp.present)/(25*3)*100),2) as present
from student_trainee_35 s
inner join studentpresent_trainee_35 sp
on s.recid = sp.recid
inner join studentdetail_trainee_35 sd
on s.recid = sd.recid
where sp.month_ in (1,2,3)
group by s.recid
having present between 60 and 75;
